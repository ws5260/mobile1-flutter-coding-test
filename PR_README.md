# 구현 중점 사항

1. MVVM 패턴을 적용
2. riverpod을 사용한 상태관리
3. Hive 패키지를 이용해 메세지 데이터 저장
4. 제공된 API를 네트워크를 통해 데이터 연동
5. 단위 테스트 코드 작성
